# Spring Boot Student Registration
A simple student registration system built with Spring Boot and MySQL, demonstrating CRUD operations, database integration, and RESTful APIs.

## Technologies Used
- Java +21
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL


#  What I Learned
- Project structuring in a real-world Spring Boot application
- Implementing RESTful APIs with proper mappings
- Connecting to a database using JPA & Hibernate

## Issues Encountered
- Database connection setup and configuration
- Implementing @PutMapping for updates
- Handling duplicate email validation
- General debugging during integration
